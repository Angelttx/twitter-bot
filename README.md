# Twitter Bot Template

Create your own Twitter bot with this template.

## Features

- 🐦 Auto-tweet
- ❤️ Auto-like
- 👥 Auto-follow
- 📊 Post analytics
- ⏰ Scheduled tweets

## Requirements

```bash
pip install tweepy
```

## Setup

1. Create Twitter Developer Account
2. Get API Keys
3. Edit `config.py`
4. Run `python bot.py`
